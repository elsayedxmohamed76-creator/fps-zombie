# FPS Zombie Game

An enhanced web-based first-person shooter zombie survival game with scoring and game over mechanics.

## How to Play

- Use **WASD** or arrow keys to move.
- Move the mouse to aim.
- Click the left mouse button to shoot.
- Survive as long as possible against waves of zombies.
- Reload automatically when ammo runs out.
- Earn 10 points for each zombie killed.
- Game ends when health reaches zero - click Restart to play again.

## Files

- `index.html`: The main HTML file.
- `css/style.css`: Styles for the game.
- `js/game.js`: The game logic and rendering.

## Running the Game

Simply open `index.html` in a web browser.

## Features

- Player movement and rotation.
- Shooting bullets with limited ammo.
- Zombie spawning and movement.
- Collision detection (bullets vs zombies, zombies vs player).
- Health and ammo display.
- Score tracking.
- Game over screen with restart functionality.
- Basic particle effects on hit.
- Automatic reload.
- Health regeneration.

Enjoy the game!
